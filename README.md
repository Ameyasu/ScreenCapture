# Win32APIで画面キャプチャ
## 概要
- 画面キャプチャの実装サンプルです
- 本アプリはデスクトップ画面をキャプチャしますがウィンドウハンドルを好きなウィンドウの物に変えたりすればそのウィンドウのキャプチャができます
- 本コードは複製・改造OKです
## 使い方
- ウィンドウが最小化されていない限り自動(30fps)で画面キャプチャし続けます
- マウスでアプリの画面内をドラッグすると画面キャプチャ位置を移動できます
- アプリの画面内でマウスが指す座標は常にRGBを表示し続けます
